นายโซลชาร์ รัตมูหมาด 6110110108 Section 02 
===============================
Application นี้พัฒนาด้วย react-native ซึ่งเป็น application สำหรับการดูสภาพอากาศ แสดงรายละเอียดต่างๆ เช่น ชื่อจังหวัด ประเทศ 
สภาพอากาศ คำอธิบายสถานะของสภาพอากศ อุณหภูมิองศา อุณหภูมิฟาเรนไฮต์ รู้สึกเหมือนกี่องศา เป็นต้น

สรุปสิ่งที่ทำตามลำดับการ Commit
Section 7.1 : เปลี่ยนภาพ Background ในแต่ละจังหวัด เปลี่ยนการแสดงในหน้า Weather จาก Zip code เป็น ชื่อจังหวัดนั้น ๆ
Section 7.2 : เพิ่มจังหวัด เพิ่มการแสดงผลองศาในหน่วยของฟาเรนไฮต์ และให้อุณหภูมิต่างๆ แสดงเพียงแค่ทศนิยม 1 ตำแหน่งให้ผู้ใช้ทราบ
Section 7.3 : เพิ่มไอคอน เพื่อแสดงสถานะของสภาพอากาศ เช่น ถ้าหาก ฝนตก ก็จะมีไอคอนเป็นรูปฝนตก เป็นต้น
Section 7.4 : เปลี่ยนหน้าเริ่มต้น Home ให้แสดงรูปภาพ ที่สื่อถึงจังหวัดตามชื่อและ Zip code
Section 7.5 : จัด Layout ตกแต่งส่วนต่าง ๆ ภายในโปรแกรมให้มี GUI ที่เหมาะสมและสวยงามมากยิ่งขึ้น
